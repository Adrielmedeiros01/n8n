# n8n - Notas:


    https://horadecodar.com.br/instalar-n8n-windows/
    
    Node.js e npm instalados
    
    node -v e npm -v
    
    npm install n8n -g
    
    Com a instalação feita, basta digitar no terminal:
    n8n
    
    Dicas rápidas de uso inicial:
    
    Deixe o terminal aberto enquanto utiliza o n8n.
    Use Ctrl + C no terminal para parar o serviço.


    5. Usando o n8n em background:
        Quer deixar o n8n rodando sem o terminal aberto? Use o comando:
        
        n8n start –tunnel
